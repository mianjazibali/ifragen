# ifragen
## Getting Started

#### Clone the repo and make it yours:

```bash
git clone git@github.com:mianjazibali/ifragen.git
cd ifragen
```

#### Install dependencies:

```bash
npm install
```

#### Set environment variables:

```bash
cp .env.example .env
```

## Running in Development

```bash
npm run dev
```

## Running in Production

```bash
npm run start
```
